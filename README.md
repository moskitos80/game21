# game21

C++ learn project realization of Black Jack console card game application for Linux platform.

## Build

Run tests:

```sh
git clone https://github.com/moskitos80/game21.git
cd game21
mkdir build
cd ./build
```

Run program:

```sh
cmake ..
make game21test
...
make test
```

In the same directory launch the game:

```sh
make Game21
...
make ./Game21
```
